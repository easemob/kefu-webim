#环信移动客服WEB

## How to BUILD

1.	Install node environment
	> https://nodejs.org/

2. 	Install gulp
	> http://gulpjs.com/

3.  Clone or Download  
	>https://github.com/easemob/web-im.git

4. 	>npm install and gulp dev

        cd ../webim
        npm install
        gulp dev

5. 	>npm install and  node app

        cd ../webim/server  
        npm install
        node app


6.  Open
	>http://localhost:8080/webim/demo.html
