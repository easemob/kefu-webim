#环信移动客服WEB

## How to BUILD

1.	Install node environment
	> https://nodejs.org/

2. 	Install gulp
	> http://gulpjs.com/

3.  Clone or Download  
	>https://github.com/easemob/web-im.git

4.  命令行进入webim文件夹 
	>npm install and gulp dev

5.  在webim/server文件夹中 
	>npm install and  node app

6.  浏览器打开
	>http://localhost:8080/webim/demo.html
